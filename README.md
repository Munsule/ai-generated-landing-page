# NovaBank Landing Page

A responsive FinTech landing page built for the NovaBank digital banking concept.

## Features

- Hero section with clear CTA
- Banking feature showcase
- Security and trust messaging
- Statistics and testimonials
- Mobile app showcase
- Download CTA and footer links

## Stack

- HTML5
- CSS3
- Responsive design using Flexbox and CSS Grid

## Usage

1. Open `index.html` in your browser.
2. Edit the page content in `index.html`.
3. Customize the styles in `style.css`.

## Repository

This repository contains:

- `index.html` — landing page markup
- `style.css` — page styling
- `ideas.txt` — NovaBank project idea notes
